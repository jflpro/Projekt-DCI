# Projet DCI - Description multilingue

---

## 🇫🇷 Français

Le **projet DCI** est une architecture réseau complète sous Linux, conçue pour simuler une infrastructure d’entreprise composée de six départements distincts. Chaque département est isolé par des VLANs et des sous-réseaux dédiés, avec un contrôle d’accès strict basé sur les rôles utilisateurs.

Ce projet intègre des fonctionnalités avancées telles que :

- Accès sécurisé via SSH et VPN  
- Pare-feu (firewall) configuré pour filtrer le trafic entre les zones  
- Gestion des super utilisateurs (Daniel, Andi, Tom, Michi, Florian, Merkito, Jeff) avec droits étendus  
- Droits restreints pour les autres utilisateurs selon leur département  
- Système de sauvegarde et redondance des données  
- Automatisation des configurations et déploiements via Ansible ou scripts Linux  
- Compatibilité avec des équipements Cisco et la virtualisation VMware pour une intégration facile en entreprise  

Ce projet vise à fournir un environnement réaliste pour la formation, le développement, la sécurité et la gestion d’un réseau d’entreprise.

## 🇬🇧 English

The **DCI project** is a complete Linux-based network architecture designed to simulate an enterprise infrastructure composed of six distinct departments. Each department is isolated by VLANs and dedicated subnets, with strict access control based on user roles.

This project includes advanced features such as:

- Secure access via SSH and VPN  
- Firewall configured to filter traffic between zones  
- Management of super users (Daniel, Andi, Tom, Michi, Florian, Merkito, Jeff) with extended rights  
- Restricted rights for other users according to their department  
- Data backup and redundancy system  
- Automation of configurations and deployments via Ansible or Linux scripts  
- Compatibility with Cisco equipment and VMware virtualization for easy enterprise integration  

This project aims to provide a realistic environment for training, development, security, and management of an enterprise network.

## 🇹🇷 Türkçe

- Ansible veya Linux betikleri aracılığıyla konfigürasyon ve dağıtım otomasyonu

Bu proje, eğitim, geliştirme, güvenlik ve kurumsal ağ yönetimi için gerçekçi bir ortam sağlamayı amaçlamaktadır.
